# 🐱 ねこ100枚 自動生成ツール

Gemini 2.5 Flash Image (Nano Banana) で統一感ある猫キャラクターを100枚自動生成します。

## 📦 含まれるファイル

- `generate-cats.mjs` - 生成スクリプト本体
- `prompts.json` - 100種類の猫のプロンプト
- `README.md` - この手順書

## 🚀 セットアップ手順

### 1. APIキーを取得（無料）

1. https://aistudio.google.com/ を開く
2. 右上「Get API key」→ 新規作成
3. キーをコピー

※ 無料枠があるので、小規模利用ならお金はかかりません

### 2. プロジェクトフォルダに配置

```
C:\Users\Jun\Downloads\あとらくしょん\Project\typing\
├─ generate-cats.mjs    ← ここに置く
├─ prompts.json         ← ここに置く
├─ .env                 ← 新規作成（次の手順）
└─ public\cats\         ← 画像がここに保存される（自動作成）
```

### 3. .env ファイル作成

プロジェクトフォルダに `.env` を新規作成し、以下を記載：

```
GEMINI_API_KEY=コピーしたAPIキーを貼り付け
```

### 4. 必要パッケージのインストール

PowerShellで：

```bash
cd C:\Users\Jun\Downloads\あとらくしょん\Project\typing
npm install @google/genai dotenv
```

### 5. 既に生成済みの4枚を配置

既にある4枚（001.png〜004.png）を `public/cats/` に配置してください。
このスクリプトは既存ファイルを自動スキップするので、5枚目から生成されます。

### 6. 実行

```bash
node generate-cats.mjs
```

## 💡 便利な使い方

### 途中で中断してもOK

Ctrl+Cで止めて、後で `node generate-cats.mjs` を再実行すれば続きから生成されます。
既に完成したファイルは自動的にスキップされます。

### 失敗したファイルだけ再生成

失敗があると `failed.txt` が作成されます。
気に入らない画像があれば、そのpngファイルを削除して再実行すれば再生成されます。

### コスト試算

- 1枚あたり：$0.039（約5.8円）
- 100枚：$3.9（約580円）
- 96枚（既に4枚ある場合）：$3.74（約560円）

## 🎨 生成後の作業

### 背景透過処理

既に生成済みの4枚のように、背景を透過したい場合：

1. https://www.remove.bg/ にアクセス
2. 画像をまとめてアップロード
3. 背景透過PNGをダウンロード

### React（ねこあつめタイピング）に組み込む

Claude Codeに以下を指示：

```
src/App.jsx の CATS 配列を修正し、各猫オブジェクトに
img: `/cats/{id:03d}.png` を追加してください。

CatFace コンポーネントは SVG ではなく <img> タグで
cat.img を表示するように変更してください。

MiniCat コンポーネントも同様に画像表示に変更してください。
```

## ❓ トラブルシューティング

### エラー: "GEMINI_API_KEY が見つかりません"

→ `.env` ファイルがスクリプトと同じフォルダにあるか確認

### エラー: "画像データがレスポンスに含まれていません"

→ 安全性フィルターにひっかかった可能性
→ プロンプトを少し変えて再実行

### レートリミットエラー

→ DELAY_MS を 5000 などに増やす（スクリプト上部で設定変更）

### 品質が気に入らない

→ そのpngを削除して再実行（ガチャ感覚で！）
